# Co-Force: Agent2Agent (A2A) Swarm Platform

Co-Force is a decentralized agent orchestration and execution platform designed for multi-node setups (e.g. Mac Mini 1 Orchestrator + Mac Mini 2 Worker Swarm) using the **Agent2Agent (A2A) Protocol** and the **Model Context Protocol (MCP)**.

For detailed documentation, requirements, architecture diagrams, and developer instructions, please visit the **[docs/](file:///Users/trungtran/ai-agents/co-force/docs/README.md)** directory.
